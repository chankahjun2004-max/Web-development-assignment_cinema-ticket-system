# Cinema Ticket System Web Assignment

## Overview

This repository contains a **Cinema Ticket System** web project created as part of a web development assignment. The application showcases a modern, responsive UI for browsing movies, viewing details, selecting seats, and completing ticket purchases.

## Features

- **Movie Catalog** – Browse a list of movies with posters, titles, and brief descriptions.
- **Movie Details** – View detailed information, trailers, and showtimes for each movie.
- **Seat Selection** – Interactive seat map with real‑time selection and visual feedback.
- **Shopping Cart** – Add tickets to a cart, view total price, and proceed to checkout.
- **Responsive Design** – Works on desktop, tablet, and mobile devices using fluid layouts and media queries.
- **Rich UI** – Utilises modern CSS techniques (gradients, glass‑morphism, subtle animations) for a premium look.

## Technology Stack

- **HTML5** – Semantic markup for structure.
- **CSS3** – Vanilla CSS with custom design system (variables, flexbox, grid, animations).
- **JavaScript (ES6+)** – Core logic for data handling, UI interactions, and dynamic content.
- **Git** – Version control.

## Setup & Development

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/chankahjun2004-max/Web-development-assignment_cinema-ticket-system.git
   cd Web-development-assignment_cinema-ticket-system
   ```
2. **Open the project** in your favorite editor (e.g., VS Code).
3. **Serve the site locally** – you can simply open any HTML file in a browser, or use a lightweight server:
   ```bash
   # Using Python's built‑in server (requires Python installed)
   python -m http.server 8000
   # Then open http://localhost:8000 in your browser
   ```
4. **Make changes** – edit HTML, CSS, or JS files. The live preview will reflect updates instantly.

## Usage

- Open `HTML FILES/Homepage1.html` in a browser to start exploring the cinema site.
- Navigate through the menu to view movies, watch trailers, and select seats.
- The cart updates automatically; click **Checkout** to simulate a purchase.
